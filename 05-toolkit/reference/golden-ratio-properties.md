# 5.1 The Golden Ratio and Its Properties

**Fundamental identity:**
$\varphi^2$ = $\varphi$ + 1 (equivalently: $\varphi$ = 1 + 1/$\varphi$)

**Powers of $\varphi$:**
$\varphi$ⁿ = F(n)·$\varphi$ + F(n−1) for all integers n (where F is Fibonacci)
$\varphi$ⁿ + $\psi$ⁿ = L(n) (Lucas numbers)
$\varphi$ⁿ − $\psi$ⁿ = F(n)·$\sqrt{5}$

**Reciprocal:**
1/$\varphi$ = $\varphi$ − 1 = 0.6180339887...
