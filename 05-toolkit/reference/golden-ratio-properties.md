# 5.1 The Golden Ratio and Its Properties

**Fundamental identity:**
φ² = φ + 1 (equivalently: φ = 1 + 1/φ)

**Powers of φ:**
φⁿ = F(n)·φ + F(n−1) for all integers n (where F is Fibonacci)
φⁿ + ψⁿ = L(n) (Lucas numbers)
φⁿ − ψⁿ = F(n)·√5

**Reciprocal:**
1/φ = φ − 1 = 0.6180339887...
