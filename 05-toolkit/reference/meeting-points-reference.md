# 5.3 The Meeting Points

$\{1, 2, 3, 5, 29, 34\}$ — complete set of integers in multiple metallic sequences.

**Metallic mean of order k:** $\mu_k = (k + \sqrt{k^2+4})/2$
- k=1: $\phi$ (golden), k=2: $\sigma$ (silver), k=3: $\beta$ (bronze)

**Metallic Fibonacci recurrence:** $M_k(n) = k \cdot M_k(n-1) + M_k(n-2)$

**Growth rates:** $\mu_k > \mu_j$ when $k > j$ (higher harmonics wind faster $\rightarrow$ sequences diverge $\rightarrow$ meeting points are finite)
