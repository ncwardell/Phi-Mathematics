# 5.3 The Meeting Points

{1, 2, 3, 5, 29, 34} — complete set of integers in multiple metallic sequences.

**Metallic mean of order k:** $\mu$_k = (k + $\sqrt{k²+4}$)/2
- k=1: $\varphi$ (golden), k=2: $\sigma$ (silver), k=3: β (bronze)

**Metallic Fibonacci recurrence:** M_k(n) = k·M_k(n−1) + M_k(n−2)

**Growth rates:** $\mu$_k > $\mu$_j when k > j (higher harmonics wind faster → sequences diverge → meeting points are finite)
