# 5.7 The Correction Factor Anatomy

The mass equation $m/m_e = \phi^d / (1 - \sigma C\alpha(1+4\alpha))$ contains:

**The self-witnessing series:**
$1/(1 - C\alpha) = 1 + C\alpha + (C\alpha)^2 + (C\alpha)^3 + ... = \Sigma(C\alpha)^n$

This is a geometric series — infinite self-witnessing. The particle must "witness itself being witnessed" through the electromagnetic channel. Each term is one more loop of self-reference.

**The $4\alpha$ correction:**
$4 = L(3) = |L(-3)|$ — the witnessing quantum coefficient. This corrects each self-witnessing loop by the hourglass geometry factor.

**The fold correction $1/(1+5\pi\alpha)$:**
$5 = F(5)$ (discriminant), $\pi$ (Hopf fiber). This additional factor for the fold ($\psi$-strand engagement) accounts for the complex structure of the $\psi$-triangle.
