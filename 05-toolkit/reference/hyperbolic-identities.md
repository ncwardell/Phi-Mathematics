# 5.4 Hyperbolic Identities at О· = ln(ПҶ)

sinh(ln ПҶ) = 1/2
cosh(ln ПҶ) = вҲҡ5/2
tanh(ln ПҶ) = 1/вҲҡ5

**General:**
e^(ln ПҶ) = ПҶ, e^(вҲ’ln ПҶ) = 1/ПҶ = ПҶвҲ’1
sinh(nВ·ln ПҶ) = (ПҶвҒҝ вҲ’ ПҶвҒ»вҒҝ)/2
cosh(nВ·ln ПҶ) = (ПҶвҒҝ + ПҶвҒ»вҒҝ)/2
