# 5.4 Hyperbolic Identities at $\eta$ = ln($\varphi$)

sinh(ln $\varphi$) = 1/2
cosh(ln $\varphi$) = $\sqrt{5}$/2
tanh(ln $\varphi$) = 1/$\sqrt{5}$

**General:**
e^(ln $\varphi$) = $\varphi$, e^(−ln $\varphi$) = 1/$\varphi$ = $\varphi$−1
sinh(n·ln $\varphi$) = ($\varphi$ⁿ − $\varphi^-$ⁿ)/2
cosh(n·ln $\varphi$) = ($\varphi$ⁿ + $\varphi^-$ⁿ)/2
