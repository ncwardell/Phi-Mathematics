# 5.4 Hyperbolic Identities at $\eta = \ln(\phi)$

$\sinh(\ln \phi) = 1/2$
$\cosh(\ln \phi) = \sqrt{5}/2$
$\tanh(\ln \phi) = 1/\sqrt{5}$

**General:**
$e^{\ln \phi} = \phi$, $e^{-\ln \phi} = 1/\phi = \phi-1$
$\sinh(n \cdot \ln \phi) = (\phi^n - \phi^{-n})/2$
$\cosh(n \cdot \ln \phi) = (\phi^n + \phi^{-n})/2$
