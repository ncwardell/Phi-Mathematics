# 5.5 The Four-Structure

{$\varphi$, $\psi$, 1, −1} where:
- $\varphi$ + $\psi$ = 1
- $\varphi$ $\times$ $\psi$ = −1
- $\varphi$ − $\psi$ = $\sqrt{5}$
- i = $\sqrt{$\varphi$ψ}$ = $\sqrt{−1}$
