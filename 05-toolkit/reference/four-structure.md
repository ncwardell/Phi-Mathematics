# 5.5 The Four-Structure

$\{\phi, \psi, 1, -1\}$ where:
- $\phi + \psi = 1$
- $\phi \times \psi = -1$
- $\phi - \psi = \sqrt{5}$
- $i = \sqrt{\phi\psi} = \sqrt{-1}$
